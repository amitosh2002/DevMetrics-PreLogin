# DevMetrics Pre-Login (React)

Pre-login + marketing pages for DevMetrics, styled with SCSS and branded with the DevMetrics logo/theme.

## Pages

- `/`: Overview (services, features, CTAs, GitHub read-only + GitLab coming soon messaging)
- `/privacy`: Privacy Policy (ported from `DevMetrics-Web`)
- `/terms`: Terms of Service
- `/post-login`: Post-login placeholder
- `/explore-tools`: Explore other tools (includes **Hora**)

## Dev

```bash
cd DevMetrics-PreLogin
npm install
npm run dev
```

## Notes

- GitHub connection is documented as the GitHub App **DevMetrics-Analytics** with **read-only** access (UI stubs only).
- GitLab is labeled as “coming soon”.
